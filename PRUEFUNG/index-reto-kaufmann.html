HTML & CSS

  1. Die CSS-Datei style.css dem Dokument hinzufügen
  2. Zwei Google Schriftarten hinzufügen. Die eine soll überall verwendet werden, die Zweite nur für Überschriften
  3. Die übergeordnete Schriftgrösse auf 1em setzen
